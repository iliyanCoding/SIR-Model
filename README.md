# SIR Model

A Jupyter notebook implementing the SIR (Susceptible-Infected-Recovered) epidemiological model to simulate the spread of infectious diseases through a population.

## Contents

- **sir.ipynb** — SIR model simulation with parameter analysis (R0, peak infection, final recovery fraction)
- **DrainingTankExample.ipynb** — ODE example of a draining tank

## Tech Stack

- Python
- NumPy & SciPy (ODE solving with `odeint`)
- Matplotlib (visualization)

## Getting Started

```bash
pip install numpy scipy matplotlib
jupyter notebook sir.ipynb
```
